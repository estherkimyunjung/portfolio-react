<br>
#####Redeployment Steps :

1. RUN `npm run build` to build updated configuration.
2. RUN `netlify deploy` to review the draft URL of your portfolio. (publish directory : `build`)
3. RUN `netlify deploy --prod` to deploy it to main site URL.
